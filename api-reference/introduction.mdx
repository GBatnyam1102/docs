## Танилцуулга (Overview)

Энэхүү API Reference нь **BNPL (Buy Now, Pay Later)** үйлчилгээний хүрээнд
хэрэглэгч, худалдаачин болон санхүүгийн байгууллагуудын хооронд
өгөгдөл солилцох **RESTful API endpoint-уудыг** тайлбарлана.

BNPL API нь дараах үндсэн үйл ажиллагааг дэмжинэ:

- Хэрэглэгчийн бүртгэл ба баталгаажуулалт
- BNPL төлбөрийн хүсэлт үүсгэх
- Зээлийн нөхцөл, хуваарь авах
- Гүйлгээний төлөв шалгах
- Төлөлтийн мэдээлэл шинэчлэх

Энэхүү хэсэг нь API-г **хэрэглэхээс өмнөх суурь ойлголтыг** өгөх зорилготой.

---

## API Баримтжуулалтын бүтэц

BNPL API Reference нь дараах байдлаар зохион байгуулагдсан:

- **Introduction** – Ерөнхий ойлголт, authentication
- **Authentication** – Нэвтрэх эрхийн механизм
- **Endpoints** – Тухайн үйлдэл бүрийн API endpoint
- **Request / Response** – Жишээ JSON өгөгдөл
- **Error Codes** – Алдааны тайлбар

> Энэхүү баримт бичиг нь **лавлагааны (reference)** зориулалттай бөгөөд  
> API хэрхэн ажилладагийг тайлбарлах биш, **яаж ашиглахыг** тодорхойлно.

---

## Authentication

BNPL API нь **endpoint-ийн зориулалтаас хамааран** хоёр төрлийн
баталгаажуулалтын механизмыг ашигладаг.

### Supported Authentication Methods

| Authentication төрөл | Ашиглагдах нөхцөл |
|---------------------|------------------|
| **Basic Authentication** | Систем хоорондын анхны итгэлцэл |
| **Bearer Token Authentication** | Хэрэглэгчийн баталгаажсан үйлдлүүд |

---
